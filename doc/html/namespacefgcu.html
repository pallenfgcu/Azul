Detailed Description

File: Azul.h.

This file represents the public interface for the FGCU mascot Azul, who with limited commands, can be made to navigate in a gridded world with simple obstacles, and can lift and place graduation caps around the board. Different scenarios loaded can offer simple to complex challenges for programming students to overcome.
